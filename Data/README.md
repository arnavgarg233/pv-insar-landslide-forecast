# Data

Small **version-controlled** assets and acquisition helpers.

| Path | Role |
|------|------|
| `figure_inputs/` | PNGs consumed by `Scripts/visualization/combine_existing_figures.py` (spatial/temporal PC panels for composite exports) |
| `acquisition/` | ASF / Sentinel-1 download helpers (ascending and descending stacks) |

Large scientific inputs (`.h5`, GPM `.nc4`, full MintPy stacks) are **not** committed; place them under the paths described in the repository `README.md` or set `LANDSLIDE_PROJECT_ROOT` accordingly.
